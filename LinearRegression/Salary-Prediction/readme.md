Salary predictor using Linear Regression
This project implements a simple salary prediction model using Linear Regression. The model is trained on a dataset containing employee experience and their corresponding salaries. The goal is to predict the salary of an employee based on their years of experience.
Dataset
The dataset used in this project is a CSV file named 'Salary_Data.csv', which contains two columns:
YearsExperience: The number of years of experience an employee has.
Salary: The salary of the employee.
The dataset can be found at: [Salary Data CSV](Salary_Dataset.csv)
